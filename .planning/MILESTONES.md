# Project Milestones: Study Buddy

## v1.0 Adaptive Learning Profiles (Shipped: 2026-01-24)

**Delivered:** Intelligent learning profile system that builds comprehensive, AI-powered profiles tracking each child's knowledge gaps, progress, and personalized next steps.

**Phases completed:** 1-5 (22 plans total)

**Key accomplishments:**

- BKT-based learner profiles with real-time mastery tracking and topic classification
- Profile-aware adaptive quiz generation with 20/50/30 difficulty mixing ratio
- AI recommendation engine with transparent reasoning and full parent override control
- Multi-signal profile integration (quiz 70%, evaluation 95%, engagement 60%, parent notes 40%)
- Ebbinghaus forgetting curves with SM-2 probe scheduling for mastery verification
- Regression detection with Hebrew alerts and 14-day cooldown per topic
- Rich analytics UI: skill radar charts, progress timelines, review mode banners
